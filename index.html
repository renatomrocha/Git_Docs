Git Docs

Local Repository

  • git init

    Initiates a git repository on local directory

  • git status

    Displays current folder status

  • git add (fold_name)

    Adds folder to local repository (add . = all files)

  • git commit -m "msg"

    Commits changes to repository

  • git log

    displays log about commits (git log --oneline for brief info)

  • git checkout (commit)(file)

    Checks files from older commits and overwrites on present repository
    (commit -> commit number from git log --oneline)

  • git checkout -- (file)

    Checks files from previous commit

  • git reset HEAD (file)

    resets file into last commit state

Remote Repository

  • git remote add (repo_name(ex:origin)) master git@github.com:username/new_repo

    Cria ligação entre repositório local e repositório remoto

  • git push -f/-u (repo_name) (branch(ex:master))

    Submete estado do repositório atual para repositório remoto

  • git clone (url)

    Clones online repository to local directory
